# Backend

Backend para gestionar la entrada y salida de elementos imprimibles por el movimiento Reesistencia.


### Requisitos

* Mysql / Sqlite (local)
* Php 7.2

## Despliege local

var directory must be writable
Copy .env.dist to .env and run symfony:
Install vendors
Run

```bash
cp .env.dist .env
composer install
symfony server:start
```

http://localhost:8000
